# Python Data Structures & Analytics Basics

Bu repozitoriyada Python proqramlaşdırma dilinin əsas məlumat strukturları (Data Structures) və formatlama metodları üzrə yerinə yetirdiyim praktiki tapşırıqlar toplanmışdır. Layihə çərçivəsində əsasən Data Analitikası üçün fundamental olan mövzular işlənilmişdir.

##  İşlənilən Mövzular və Tapşırıqlar:

*   **Variables & Math Operations:** Dəyişənlərin təyini, riyazi hesablamalar və tip çevrilmələri (`int`, `float`).
*   **String Manipulation:** Mətnlərin kəsilməsi (slicing), uzunluq hesabı (`len()`), sözlərin əvəzlənməsi (`.replace()`) və indeks tapılması (`.find()`).
*   **F-string Formatting:** Ədədlərin fərqli onluq formatlarda (1f, 2f, 3f) oxunaqlı şəkildə ekrana çıxarılması.
*   **Lists (Siyahılar):** Siyahıların əlifba sırası ilə sıralanması (`.sort()`), sonuna element əlavə edilməsi (`.append()`), elementlərin silinməsi (`.pop()`) və istənilən indeksə yerləşdirilməsi (`.insert()`).
*   **Text & List Conversion:** Mətnin `.split()` ilə parçalanaraq siyahıya çevrilməsi və `.join()` metodu ilə yenidən mətn halına gətirilməsi.
*   **Dictionaries (Lüğətlər):** Lüğətlərdə açar (key) və dəyərlərlə (value) iş, məlumatların yenilənməsi və oxunması.
*   **Tuples & Sets:** Dəyişdirilə bilməyən (`immutable`) məlumat tipləri ilə iş və təkrarlanan elementləri qəbul etməyən `set` (çoxluq) strukturu.
